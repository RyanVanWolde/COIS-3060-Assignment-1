This is all the available data sources.

Source Name Data Type(s) Description
Github Repositories, commits, pull requests, issues, and stars GitHub is a cloud-based platform that hosts Git repositories, allowing developers to store, manage, track, and share code securely for open-sourced projects.
Stack Overflow Questions, answers, tags, and user reputation Stack Overflow is a popular question-and-answer community forum for programmers to assit eachother in solving errors.
JIRA Issue tracking, sprint data, and bug reports JIRA is a project management tool used by software teams; designed to help teams plan, track, and manage work, particularly using Agile methodologies.
App Stores Google Play and Apple App Store reviews, ratings, and downloads App Stores (specifically Google Play and the Apple App Store) are marketplaces where users can download and review applications to their devices.
Travis CI / Jenkins Build logs, test results, and deployment data Travis CI / Jenkins are hosted continuous integration (CI) and continuous delivery (CD) services used to automatically build, test, and deploy software projects.
Mailing Lists Developer discussions, and archives Mailing Lists are collections of names and email addresses used by individuals or organizations to send messages, newsletters, or updates to multiple recipients simultaneously.
npm Registry Package dependencies, downloads, and versions npm Registry is a large, public online database of JavaScript software packages and their metadata, serving as the central hub for the JavaScript and Node.js open-source ecosystem.
