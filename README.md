# COIS-3060-Assignment-1
A webpage for MSR Research Group

### Project Description
The MSR Research Group Webpage is an online research guide for students and researchers studying mining software repositories. On the website, there is information to aid in understanding software development practices, such as data sources, collection methods, analysis methods, and publication venues. With this design, people can easily access the informaiton they need in one place, saving on research time in a one-stop-shop.
