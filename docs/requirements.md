# User Stories

## User Story 1:
As a researcher, when I am studying software development practices I want to be to see all available data sources and get a cursory glance at what types of data each contains.

## User Story 2: 
As a PhD student, I want to be able to see how we collect data, so I can see which works is compatible with my project.

## User Story 3:
As a researcher, I want to be able to see all the data collection methods and their pros/cons, so I can compare which would best suit me.

## User Story 4:
As a researcher, I want the webpage to look professional and clean so that I can reccomend it to colleagues as a reputable source.

## User Story 5: 
As a new student, I want the website to be easy to naviagte, since some sources in academia are confusing and I want an easy and quick look at the information I need.
